# 扩散模型核心直觉

> 🔰 入门 | 本文是 Diffusion 系列的第一篇，不需要任何扩散模型的前置知识。你只需要了解基本的深度学习概念（神经网络、损失函数、梯度下降）。

## 一滴墨水的故事

想象你往一杯清水中滴入一滴墨水。墨水会逐渐扩散，从一个集中的墨点变成均匀的淡色——这个过程是自发的、不可逆的。最终，你无法分辨墨水最初滴在哪里。

现在想象你能"倒放"这个过程：从一杯均匀的淡墨水出发，让墨水逐步聚集，最终还原成那个清晰的墨点。

**扩散模型（Diffusion Model）** 正是基于这个直觉：

1. **前向过程（Forward Process）**：向数据逐步添加噪声，就像墨水扩散——数据从"有结构"变成"纯噪声"
2. **反向过程（Reverse Process）**：训练一个神经网络学习如何逐步去除噪声，就像"倒放"扩散——从"纯噪声"恢复出"有结构"的数据

一旦学会了"倒放"，我们就可以从随机噪声出发，逐步去噪，**生成**全新的、逼真的数据（图像、视频、音频等）。

## 前向过程：逐步加噪

前向过程非常简单，不需要学习任何参数。我们只是反复地向数据添加少量**高斯噪声（Gaussian Noise，即服从正态分布的随机噪声）**：

```
原始图像 x₀ → 略有噪声 x₁ → 更多噪声 x₂ → ... → 纯噪声 xₜ
```

每一步添加的噪声量由一个预定义的**噪声调度表（Noise Schedule）** 控制。经过足够多的步数（通常 $T = 1000$ 步），原始数据会被完全淹没在噪声中，变成与标准高斯分布几乎无法区分的纯随机噪声。

关键洞察：虽然每一步的变化很小，但累积效果会完全摧毁原始数据中的所有信息。

## 反向过程：逐步去噪

反向过程是扩散模型的核心——我们训练一个**去噪神经网络（Denoising Neural Network）** $\epsilon_\theta$，让它学会在每一步预测并去除噪声：

```
纯噪声 xₜ → 去噪一步 xₜ₋₁ → 再去噪 xₜ₋₂ → ... → 干净数据 x₀
```

训练的方式出人意料地简单：

1. 取一张干净图像 $x_0$
2. 随机选一个时间步 $t$，直接加噪到 $x_t$（有数学捷径，不需要逐步加噪）
3. 让网络看着 $x_t$ 和时间步 $t$，预测加入的噪声 $\epsilon$
4. 用预测噪声与真实噪声之间的**均方误差（Mean Squared Error, MSE）** 作为损失函数
5. 梯度下降，更新网络参数

训练好之后，生成新数据只需要：从纯随机噪声 $x_T$ 开始，反复调用去噪网络，逐步得到干净数据 $x_0$。

## 一个直觉类比

你可以把去噪网络想象成一个"雕塑家"：

- 纯噪声 $x_T$ 是一块粗糙的大理石
- 每一步去噪就是雕塑家削掉一点多余的石头
- 经过 $T$ 步精心雕刻，一件精美的雕塑（生成的图像）就诞生了

不同的初始噪声（不同的大理石）会产生不同的作品——这就是扩散模型能够生成多样化结果的原因。

## 与其他生成模型的对比

在扩散模型之前，已有几种主流的**生成模型（Generative Model，即学习数据分布并生成新样本的模型）**。下面是它们的核心思想对比：

### GAN（Generative Adversarial Network，生成对抗网络）

**核心思想**：两个网络对抗博弈——生成器（Generator）负责造假，判别器（Discriminator）负责鉴别真假。

```
随机噪声 z → 生成器 → 生成图像
                         ↕ 对抗
                      判别器 ← 真实图像
```

| 优势 | 劣势 |
|------|------|
| 生成速度快（单次前向传播） | 训练不稳定（模式崩塌、训练震荡） |
| 生成质量高（曾经的 SOTA） | 多样性不足（容易丢失数据模式） |
| | 难以评估和控制生成过程 |

### VAE（Variational Autoencoder，变分自编码器）

**核心思想**：将数据编码到一个**潜空间（Latent Space，即低维的连续表示空间）**，从潜空间采样再解码生成新数据。

```
数据 x → 编码器 → 潜变量 z ~ q(z|x)
潜变量 z → 解码器 → 重建数据 x̂
```

| 优势 | 劣势 |
|------|------|
| 训练稳定 | 生成图像往往模糊 |
| 有明确的概率框架 | 潜空间的表达能力受限 |
| 支持插值和编辑 | 生成质量通常低于 GAN |

### Normalizing Flow（标准化流）

**核心思想**：通过一系列可逆变换，将简单分布（如高斯分布）精确映射到复杂的数据分布。

```
简单分布 z → f₁ → f₂ → ... → fₖ → 数据 x
（每个 fᵢ 都是可逆的，可以精确计算概率）
```

| 优势 | 劣势 |
|------|------|
| 可以精确计算似然（Likelihood） | 模型必须可逆，架构受限 |
| 训练稳定 | 计算量大 |
| | 生成质量有时不如 GAN |

### Diffusion Model（扩散模型）

```
数据 x₀ → 加噪 → x₁ → ... → xₜ (纯噪声)
纯噪声 xₜ → 去噪 → xₜ₋₁ → ... → x₀ (生成数据)
```

| 优势 | 劣势 |
|------|------|
| 训练稳定（简单的 MSE 损失） | 采样速度慢（需要多步去噪） |
| 生成质量极高（当前 SOTA） | 计算量大（训练和推理） |
| 天然支持条件生成和引导 | |
| 多样性好（覆盖数据分布的各种模式） | |

## 为什么 Diffusion 能胜出？

扩散模型在 2020 年（DDPM 论文）后迅速崛起，到 2022-2023 年已全面超越 GAN 成为图像生成的主流方法。核心原因有三：

1. **训练稳定性**：扩散模型的训练目标是简单的去噪回归（MSE 损失），不存在 GAN 的对抗训练不稳定问题。这意味着更容易复现、更容易扩展（Scale up）。

2. **多样性与质量的平衡**：GAN 容易出现**模式崩塌（Mode Collapse，即生成器只学会生成少数几种样本，忽略数据分布的其他模式）**。扩散模型通过逐步去噪覆盖整个数据分布，天然具有更好的多样性。

3. **灵活的条件控制**：通过**引导（Guidance）** 技术（后续章节详述），扩散模型可以轻松地用文本、图像、语义图等条件来控制生成，这是 Stable Diffusion、DALL·E 等产品背后的关键能力。

## 小结

| 概念 | 一句话解释 |
|------|-----------|
| 扩散模型 | 通过"加噪→去噪"的过程学习生成数据 |
| 前向过程 | 逐步向数据添加噪声，直到变成纯随机噪声 |
| 反向过程 | 训练神经网络逐步去噪，从噪声恢复/生成数据 |
| 训练目标 | 预测每一步添加的噪声（MSE 损失） |
| 生成流程 | 从随机噪声出发，反复去噪得到生成结果 |
