# 超分辨率与图像修复

> ⚙️ 进阶 | 前置知识：[DDPM 详解](../02-models-zoo/01-ddpm.md)，[条件生成概述](../03-conditional-generation/01-overview.md)

## Diffusion 在底层视觉任务中的优势

传统**超分辨率（Super-Resolution, SR）** 和**图像修复（Image Restoration）** 方法使用 CNN 回归模型（如 EDSR、RCAN），通过 L1/L2 损失直接回归高分辨率/干净图像。这种方法存在根本性缺陷：**回归目标的多义性**。

### 回归方法为什么会模糊？

对于同一张低分辨率输入，存在无穷多个合理的高分辨率结果（例如模糊区域的纹理细节可以有多种合理选择）。L2 损失驱动模型输出所有可能结果的**条件均值**——在像素空间做平均的结果就是**模糊**。

```
                  同一张低分辨率输入
                         │
            ┌────────────┼────────────┐
            ▼            ▼            ▼
      合理结果 A    合理结果 B    合理结果 C
     (纹理方案1)   (纹理方案2)   (纹理方案3)
            │            │            │
            └────────┬───┘────────────┘
                     ▼
              L2 损失 → 输出均值
            ┌─────────────────┐
            │   模糊的结果    │  ← 三种纹理的平均 = 没有纹理
            └─────────────────┘
```

Diffusion 模型的根本优势在于它是**生成式方法**——从条件分布 $p(x_{\text{HR}} | x_{\text{LR}})$ 中**采样**，而非输出均值。每次采样给出一个特定的、细节丰富的结果。

## 条件扩散超分辨率：数学框架 🔬

### 基本公式

超分辨率的条件扩散建模：

$$p_\theta(x_{\text{HR}}^{0:T} | x_{\text{LR}}) = p(x_{\text{HR}}^T) \prod_{t=1}^{T} p_\theta(x_{\text{HR}}^{t-1} | x_{\text{HR}}^t, x_{\text{LR}})$$

关键：前向过程只对高分辨率图像 $x_{\text{HR}}$ 加噪，低分辨率图像 $x_{\text{LR}}$ 作为条件始终保持干净。

训练目标（条件噪声预测）：

$$\mathcal{L} = \mathbb{E}_{x_{\text{HR}}, x_{\text{LR}}, \epsilon, t} \left[ \| \epsilon - \epsilon_\theta(x_{\text{HR}}^t, t, x_{\text{LR}}) \|^2 \right]$$

### 条件注入方式

| 方式 | 做法 | 优势 | 使用模型 |
|------|------|------|---------|
| **通道拼接** | $[\text{Upsample}(x_{\text{LR}}); x_t^{\text{HR}}]$ → U-Net | 简单直接，低分辨率信息在每个空间位置都可见 | SR3, Imagen SR |
| **交叉注意力** | $x_{\text{LR}}$ 特征通过 Cross-Attention 注入 | 灵活，适合跨模态条件 | LDM-SR |
| **编码器注入** | $x_{\text{LR}}$ 编码后注入 U-Net skip connections | 多尺度特征融合 | StableSR |

## SR3：Diffusion 超分辨率的开创性工作 ⚙️

### SR3 架构

**SR3（Super-Resolution via Repeated Refinement）**（Saharia et al., 2022）使用最简单的通道拼接方式注入低分辨率条件：

```
SR3 架构: x_LR → Bicubic 上采样 → [上采样LR(3ch) | 噪声x_t(3ch)] = 6ch
  → U-Net(时间步嵌入+ResBlock+Self-Attn+编解码器) → 预测噪声ε → DDPM去噪 → x_HR
```

### SR3 实验结果

在 CelebA-HQ 面部 16×16 → 128×128（8× 超分）任务上：

| 方法 | PSNR↑ | SSIM↑ | FID↓ | 人类偏好（混淆率） |
|------|-------|-------|------|-----------------|
| FSRGAN | 24.04 | 0.63 | 37.8 | 19.8% |
| PULSE | 20.78 | 0.54 | 42.4 | 11.2% |
| SR3 | 23.22 | 0.59 | **5.2** | **50.2%** |
| 真实图像 (reference) | ∞ | 1.0 | 0 | 50.0% |

**惊人发现**：SR3 的人类混淆率达到 50.2%——评测者无法区分 SR3 输出和真实图像！尽管 PSNR 低于 FSRGAN，但感知质量远超。这证实了 PSNR 作为评估指标的局限性。

## PSNR vs LPIPS：失真与感知的权衡 🔬

### 两个指标的本质矛盾

| 指标 | 全称 | 衡量内容 | 越高/低越好 | 偏好特性 |
|------|------|---------|-----------|---------|
| **PSNR** | Peak Signal-to-Noise Ratio | 像素级精确度 | 越高越好 | 偏好模糊但"安全"的均值输出 |
| **LPIPS** | Learned Perceptual Image Patch Similarity | 感知相似度 | 越低越好 | 偏好清晰自然的细节纹理 |

```
                失真-感知权衡曲线 (distortion-perception trade-off)
LPIPS (感知质量)
  │
0.5├── EDSR ●                                      ← 模糊但 PSNR 高
  │
0.4├────── RCAN ●
  │
0.3├──────── RealESRGAN ●
  │
0.2├──────────── SR3 ●                             ← 清晰自然
  │
0.1├────────────── SDUpscaler ●                    ← 细节丰富
  │
  └──┬───┬───┬───┬───┬───┬───┬───── PSNR (dB)
    22  23  24  25  26  27  28  29

    ◄───── 感知质量好 ─────────────── 像素精确 ─────▶
```

**关键洞察**：Blau & Michaeli (2018) 从理论上证明了**失真与感知质量之间存在不可避免的权衡（trade-off）**。不可能同时最优化两者——追求 PSNR 必然牺牲感知质量，反之亦然。Diffusion SR 方法选择了感知质量优先的路线。

### 典型数值对比（Set14 4× SR）

| 方法 | 类型 | PSNR↑ (dB) | SSIM↑ | LPIPS↓ | FID↓ | 推理时间 |
|------|------|-----------|-------|--------|------|---------|
| EDSR | CNN 回归 | **28.80** | **0.788** | 0.270 | 48.2 | ~0.05s |
| RCAN | CNN 回归 | 28.87 | 0.790 | 0.265 | 46.1 | ~0.08s |
| RealESRGAN | GAN | 26.28 | 0.741 | 0.124 | 25.3 | ~0.1s |
| SwinIR | Transformer 回归 | 28.94 | 0.791 | 0.258 | 44.5 | ~0.3s |
| SR3 | Diffusion | 25.60 | 0.680 | **0.072** | **8.1** | ~60s |
| StableSR | Latent Diffusion | 26.15 | 0.715 | 0.082 | 10.5 | ~5s |

## Stable Diffusion Upscaler：潜空间超分 ⚙️

### 管线设计

Stability AI 发布的 SD x4 Upscaler 将 SR 带入潜空间：

```
低分辨率图 (128×128)
     │
     ▼ Bicubic 上采样
     │
上采样图 (512×512)
     │
     ▼ 添加可控噪声 (noise augmentation, level ∈ [0, 200])
     │
     ▼ VAE 编码
     │
潜变量条件 (64×64×4)
     │
     ▼ 与噪声潜变量拼接 → 8 通道
     │
U-Net 去噪 (条件: 文本 prompt + 低分辨率潜变量)
     │
     ▼ VAE 解码
     │
高分辨率输出 (512×512)
```

**噪声增强（Noise Augmentation）**：对上采样的低分辨率图像添加少量噪声，使模型对输入中的伪影（JPEG 压缩、模糊）更鲁棒。噪声级别作为额外条件注入 U-Net。

## 级联超分辨率：Imagen 的方法 🔬

Imagen 使用两级超分辨率将 64×64 基础生成提升到 1024×1024：

```
Stage 1: 文本 → 64×64           Stage 2: 64→256                Stage 3: 256→1024
┌─────────────────────┐        ┌─────────────────────┐        ┌─────────────────────┐
│ Base Model (2B)     │        │ SR Model 1 (600M)   │        │ SR Model 2 (400M)   │
│ T5-XXL 条件         │ ──────▶│ 通道拼接条件         │ ──────▶│ 通道拼接条件         │
│ DDPM 1000 步        │        │ + noise augment     │        │ + noise augment     │
│ CFG w=8.0           │        │ DDPM 1000 步        │        │ DDPM 1000 步        │
└─────────────────────┘        └─────────────────────┘        └─────────────────────┘

关键技巧：
- 每级 SR 模型对输入添加随机噪声 (noise augmentation)
- 噪声级别作为条件注入，使 SR 模型学会处理不同质量的输入
- 这使得 Stage 1 的小缺陷不会被 Stage 2/3 放大
```

### 级联 vs 直接 vs 潜空间上采样

| 策略 | 代表 | 总参数 | 优势 | 劣势 |
|------|------|--------|------|------|
| 级联 (64→256→1024) | Imagen | ~3B (三模型) | 每级计算量小，高分细节好 | 需要训练三个模型，推理三次 |
| 直接高分辨率 | SDXL | 2.6B | 单模型端到端 | 训练成本极高 |
| 潜空间上采样 | SD x4 Upscaler | ~1.7B | 复用预训练 SD | 两阶段推理 |

## 盲超分辨率：统一退化模型 🔬

实际应用中，低分辨率图像的退化方式未知（可能是下采样+JPEG 压缩+噪声+模糊的任意组合）。**盲超分辨率（Blind SR）** 需要模型处理未知退化。

### 真实世界退化模型（Real-ESRGAN 风格）

对高分辨率图像施加**两阶段退化**：每阶段包含随机模糊核（Gaussian/Sinc, $\sigma \in [0.2, 3]$）→ 随机下采样 → 随机噪声（Gaussian $\sigma \in [1,30]$ / Poisson）→ JPEG 压缩（quality $\in [30, 95]$）。二阶退化模拟真实世界图像经过多次处理的效果。

将这种统一退化模型与 Diffusion SR 结合（如 StableSR、DiffBIR），可以训练出处理各种真实世界低质量图像的盲超分辨率模型。

## 广义图像修复：统一条件扩散框架

Diffusion 的条件生成框架可以将多种底层视觉任务统一为：**给定退化观测 $y$，从后验分布 $p(x|y)$ 中采样干净图像 $x$**。

### 统一公式

退化模型：$y = \mathcal{A}(x) + n$，其中 $\mathcal{A}$ 是退化算子，$n$ 是噪声。

| 任务 | 退化算子 $\mathcal{A}$ | 条件输入 $y$ | 输出 $x$ |
|------|----------------------|-------------|---------|
| **超分辨率** | 下采样算子 $\mathcal{D}_s$ | 低分辨率图 | 高分辨率图 |
| **去噪** | 恒等 $\mathcal{I}$ | 噪声图 | 干净图 |
| **去模糊** | 模糊核卷积 $k * \cdot$ | 模糊图 | 清晰图 |
| **JPEG 去伪影** | JPEG 压缩 $\mathcal{J}_q$ | 压缩图 | 无伪影图 |
| **补全 (Inpainting)** | 遮罩 $M \odot \cdot$ | 不完整图 + 遮罩 | 完整图 |
| **着色 (Colorization)** | 灰度化 $\mathcal{G}$ | 灰度图 | 彩色图 |
| **HDR 重建** | 色调映射 $\mathcal{T}$ | LDR 图 | HDR 图 |

### 无训练后验采样方法

不需要为每个任务专门训练模型！利用预训练的无条件扩散模型 $p_\theta(x)$ 和已知退化算子 $\mathcal{A}$，通过后验采样近似 $p(x|y)$：

```python
# DDRM / DPS / DDNM 等方法的通用框架
def posterior_sampling(model, y, A, num_steps=1000):
    """
    model: 预训练的无条件扩散模型
    y:     退化观测
    A:     已知退化算子（可微分）
    """
    x_t = torch.randn_like(target_shape)  # 从纯噪声开始
    
    for t in reversed(range(num_steps)):
        # Step 1: 无条件去噪一步
        x_0_hat = model.predict_x0(x_t, t)
        
        # Step 2: 数据一致性修正
        # 使 A(x_0_hat) 接近观测 y
        consistency_grad = (A(x_0_hat) - y)  # 数据不一致度
        x_0_hat = x_0_hat - step_size * A_pseudoinverse(consistency_grad)
        
        # Step 3: 加噪到 t-1
        x_t = ddpm_step(x_t, x_0_hat, t)
    
    return x_t
```

**优势**：一个预训练模型 + 不同退化算子 = 解决所有修复任务，无需针对每个任务训练。
**代表工作**：DDRM（Kawar et al., 2022）、DPS（Chung et al., 2023）、DDNM（Wang et al., 2023）。

## 与传统 CNN 方法的综合对比

| 维度 | CNN 回归 (EDSR/SwinIR) | GAN (RealESRGAN) | Diffusion (SR3/StableSR) |
|------|----------------------|-----------------|------------------------|
| 输出类型 | 确定性（单一结果） | 确定性（单一结果） | 随机性（每次不同） |
| PSNR | 最高 | 中 | 较低 |
| LPIPS (感知) | 差 | 好 | 最好 |
| FID | 差 | 好 | 最好 |
| 主观质量 | 模糊 | 较清晰但有伪影 | 最自然 |
| 推理速度 | 极快（~0.05s） | 快（~0.1s） | 慢（5-60s） |
| 训练稳定性 | 稳定 | 需要小心调参 | 稳定 |
| 多样性 | 无 | 无 | 有（同一输入多种结果） |
| 退化鲁棒性 | 需要针对性训练 | Real-ESRGAN 较好 | 结合退化模型后很好 |

## 小结

| 概念 | 要点 |
|------|------|
| 回归 vs 生成 | L2 回归输出条件均值→模糊；Diffusion 从分布采样→清晰自然 |
| PSNR vs LPIPS | 存在不可避免的 trade-off，Diffusion 选择感知优先 |
| SR3 | 通道拼接条件注入，人类混淆率 50%（无法区分真假） |
| 级联 SR | Imagen 路线：64→256→1024，噪声增强防止错误级联 |
| 盲 SR | 统一退化模型模拟真实世界多种退化的组合 |
| 统一框架 | 所有修复任务 = 条件后验采样 $p(x|y)$，一个模型解决所有任务 |
