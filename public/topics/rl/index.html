RL 核心概念速查 / Core Concept Map

强化学习的三大范式与代表算法对照

Value-Based / 基于价值

学习最优价值函数，隐式推导策略

  • Q-Learning / DQN
  • Double DQN、PER、Replay Buffer
  • Dueling DQN、Distributional RL
  • Rainbow (all combined)

Policy-Based / 基于策略

直接参数化策略，用梯度上升优化

  • REINFORCE（Monte Carlo PG）
  • Actor-Critic（TD PG）
  • A2C / A3C（同步/异步）
  • ACKTR（Kronecker 近似）

Model-Based / 基于模型

学习环境转移模型，规划行为

  • Dyna、World Models
  • Dreamer (RSSM)
  • MuZero、AlphaZero
  • MBPO、Sim2Real
