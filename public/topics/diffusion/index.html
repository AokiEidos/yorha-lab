扩散模型核心概念速查 / Core Concept Map

扩散模型的三大阶段与代表算法对照

Forward Process / 加噪

固定 Markov 链，逐步向数据添加高斯噪声

  • $q(x_t \mid x_{t-1}) = \mathcal{N}(\sqrt{1-\beta_t}x_{t-1}, \beta_t\mathbf{I})$
  • 闭式解：$q(x_t \mid x_0) = \mathcal{N}(\sqrt{\bar{\alpha}_t}x_0, (1-\bar{\alpha}_t)\mathbf{I})$
  • 噪声调度：Linear / Cosine / Sqrt / LogSNR

Reverse Process / 去噪

学习条件分布，从噪声逐步重建数据

  • $p_\theta(x_{t-1} \mid x_t)$：UNet / DiT 预测噪声 $\epsilon_\theta$
  • 简化损失：$\mathbb{E}[\|\epsilon - \epsilon_\theta\|^2]$
  • 参数化：Noise / $x$ / $v$-Prediction

条件控制 / Conditioning

通过条件注入实现受控生成

  • Cross-Attention：文本/图像 → latent
  • CFG：$\hat{\epsilon} = \epsilon_w + s(\epsilon_w - \epsilon_\emptyset)$
  • ControlNet：Edge/Pose/Depth 图控制

扩散模型标准流程 / Standard Diffusion Pipeline

真实数据 $x_0$
（图像/视频/BEV）
→
Forward $q$
逐步加噪 $x_0 \to x_T$
→
纯噪声 $x_T$
（$T=1000$）
$x_T \sim \mathcal{N}(0,\mathbf{I})$
→
Reverse $p_\theta$
UNet/DiT 去噪 $x_T \to x_0$
→
生成数据 $\hat{x}_0$
（新图像/视频/BEV）

架构演进

2020 — DDPM

UNet + noise prediction，ImageNet 256² FID 3.7

2022 — LDM (SD)

VAE latent space + cross-attention，文本生图普及

2023 — DiT

Transformer 替代 UNet，Scaling Law 成立

2024 — SD3 / FLUX

MM-DiT，多模态条件融合，文本质量大幅提升

采样加速

2021 — DDIM

非马尔可夫反向过程，$T=50$ 步，10-20× 加速

2022 — DPMSolver

高阶 ODE 求解器，$T=20$ 步可达满意质量

2023 — LCM

Latent Consistency Model，$T=4$ 步，蒸馏无需 CG

2024 — SDXL-Turbo

对抗蒸馏，$T=1-4$ 步，实时生图成为可能
