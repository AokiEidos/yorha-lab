03 反向过程与DDPM目标

反向过程与DDPM目标

反向过程定义 / Reverse Process Definition

扩散模型的本质：学习从噪声到数据的条件分布

学习的条件分布

Forward Process 是已知的（固定的加噪），Reverse Process 是需要学习的。我们用一个参数化神经网络 $p_\theta(\cdot)$ 来近似真实的反向过程 $q(\cdot)$：

$$p_\theta(x_{t-1} \mid x_t) = \mathcal{N}\Bigl(x_{t-1}; \mu_\theta(x_t, t), \sigma_t^2 \mathbf{I}\Bigr)$$

其中 $\mu_\theta(x_t, t)$ 是神经网络预测的均值，$\sigma_t$ 是与噪声调度相关的标准差。这个高斯分布的参数全部由 $\theta$（UNet/DiT 的权重）决定。

贝叶斯后验

给定 $x_0$ 和 $x_t$，我们可以精确计算后验 $q(x_{t-1} \mid x_t, x_0)$：

$$q(x_{t-1} \mid x_t, x_0) = \mathcal{N}\Bigl(x_{t-1}; \tilde{\mu}_t(x_t, x_0), \tilde{\sigma}_t^2 \mathbf{I}\Bigr)$$

其中 $\tilde{\mu}_t = \frac{\sqrt{\bar{\alpha}_{t-1}}\beta_t}{1-\bar{\alpha}_t}x_0 + \frac{\sqrt{\alpha_t}(1-\bar{\alpha}_{t-1})}{1-\bar{\alpha}_t}x_t$。这就是训练时使用的目标分布。

为什么是高斯分布？
当 $\beta_t$ 足够小时，$q(x_{t-1} \mid x_t)$ 可以被证明近似为高斯分布（中心极限定理）。无论数据分布多复杂，多步小噪声 Markov 链的反向过程仍然是高斯的。

为什么需要反向过程 / Why Learn the Reverse Process

正向加噪容易，但直接从噪声生成数据需要学习——反向过程是扩散模型的核心

Forward Process 的已知性

Forward Process（加噪）是完全已知、无需学习的：

$$q(x_t \mid x_{t-1}) = \mathcal{N}\bigl(x_t; \sqrt{1-\beta_t}\, x_{t-1},\, \beta_t \mathbf{I}\bigr)$$

给定任意 $x_0$，我们可以直接算出任意 $x_t$ 的分布（闭式解）。加噪过程不需要参数，完全由噪声调度 $\{\beta_t\}$ 决定。

关键问题：反向呢？
$q(x_{t-1} \mid x_t)$ 才是我们需要建模的分布。理论上，这是真实数据分布的函数——我们不知道，只能用神经网络去近似。

直接建模的困难

如果我们直接建模 $p_\theta(x_0)$（数据的生成分布）：

❌ 数据分布复杂

自然图像的分布是高维、非结构化的，直接用神经网络拟合需要超大规模参数

✅ 条件高斯更好

$q(x_{t-1} \mid x_t)$ 在小噪声时近似高斯分布，每步只需预测均值和方差

反向过程的每一步都是从一个高斯分布到另一个高斯分布的转换——这远比直接建模整个数据分布简单。每一步只需要预测一个小的偏移，而不是整个数据空间的结构。

多步累积的优势
反向过程每次只走一小步（$\beta_t$ 很小），这使得每一步的条件分布足够简单。随着 $T$ 步的累积效果，最终能从纯噪声生成分布良好的数据。类似思想在 GAN 的逐步生成中也有体现。

变分下界（ELBO）/ Evidence Lower Bound

如何从 $p_\theta(x_0)$ 的对数似然推导出可优化的目标函数

KL 散度目标推导

对数似然 $\log p_\theta(x_0)$ 可以分解为：

$$\log p_\theta(x_0) = \mathcal{L}_0 + \sum_{t=1}^{T-1} \mathcal{L}_t + \mathcal{L}_T$$

其中每项 $\mathcal{L}_t = D_{\mathrm{KL}}\bigl(q(x_t \mid x_{t-1}, x_0) \Vert p_\theta(x_t \mid x_{t-1})\bigr)$。这就是 VLB（Variational Lower Bound）。在 DDPM 中，$q(x_t \mid x_{t-1}, x_0) = q(x_t \mid x_{t-1})$（条件独立性），因此 KL 散度简化为直接比较两个高斯分布——有解析解！

DDPM Simple Loss（简化损失）

Ho et al. (2020) 发现：直接预测加入的噪声 $\epsilon$ 比预测均值更简单，效果更好：

$$\mathcal{L}_{\text{simple}} = \mathbb{E}_{t, x_0, \epsilon}\Bigl[ \| \epsilon - \epsilon_\theta(x_t, t) \|^2 \Bigr]$$

其中 $x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1-\bar{\alpha}_t} \epsilon$，$\epsilon \sim \mathcal{N}(0, \mathbf{I})$。这个损失在 $t$ 上均匀采样。ImageNet 256×256 的实验证明：简化损失比原始 ELBO 收敛更快、生成质量更高。

简化损失的物理直觉
预测噪声等价于预测"偏离真值的方向"。在 $\bar{\alpha}_t$ 接近 0.5（$t\approx 500$）时，$\epsilon$ 和 $x_0$ 的权重相近，模型需要同时理解语义和细节——这是最难的任务，但也是最有判别性的。

三种参数化方式 / Three Parameterizations

神经网络预测什么？噪声 $\epsilon$、原始数据 $x_0$、还是速度 $v$？

对比表

参数化预测目标优缺点
Noise Prediction噪声 $\epsilon$DDPM 默认，最稳定，训练收敛快，与 DDIM 兼容性好
$x$-Prediction原始数据 $x_0$直接生成 $x_0$，但数值不稳定（$1/\sqrt{\bar{\alpha}_t}$ 在 $t\to 0$ 时很大）
$v$-Prediction速度向量 $v$Stable Diffusion 2+ 采用，$t\to 0$ 时更稳定， $\epsilon$ 和 $x_0$ 的加权平均
$$v = \sqrt{\bar{\alpha}_t} \cdot \epsilon - \sqrt{1-\bar{\alpha}_t} \cdot x_0$$

$v$-Prediction 预测的是 $\epsilon$ 和 $x_0$ 的加权差，在 $t$ 接近 0 时方差更小，训练更稳定。

实用建议
图像生成任务优先尝试 $v$-prediction（SD2+ 默认），收敛更稳定。Noise prediction 是最通用的选择，与 SDXL/LCM 的蒸馏目标兼容更好。$x$-prediction 在 $T$ 很小（如 $T\leq 10$）时可能更好。

训练与采样算法 / Training and Sampling

DDPM 训练

# DDPM 训练算法（Algorithm 1）
while not converged:
    x0 ~ q(x0)         # 从真实数据采样
    t ~ Uniform({1, ..., T})  # 随机时间步
    eps ~ N(0, I)     # 采样噪声
    
    # 加噪：利用闭式解
    xt = sqrt(bar_alpha[t]) * x0 + \
         sqrt(1 - bar_alpha[t]) * eps
    
    # 梯度下降：简化损失
    loss = ||eps - eps_theta(xt, t)||^2
    theta = theta - lr * grad(loss)

DDPM 采样

# DDPM 采样算法（Algorithm 2）
xT ~ N(0, I)        # 从纯噪声开始
for t = T, T-1, ..., 1:
    z ~ N(0, I) if t > 1 else z = 0
    mu = mean(eps_theta(x_t, t))
    alpha_t = 1 - beta_t
    sigma_t = sqrt(beta_t)
    x_{t-1} = (xt - beta_t / \
               sqrt(1-bar_alpha[t]) * \
               mu) / sqrt(alpha_t) + \
               sigma_t * z
采样复杂度分析
DDPM 采样需要 $T$ 次神经网络评估（1000步），每次都是完整的 UNet/DiT 前向传播。在 $T=1000$ 时，采样延迟约为 10-30 秒（取决于硬件）。这个延迟是扩散模型最大的实用瓶颈，也是加速采样研究的根本动机。

DiT 中的反向过程 / Reverse Process in DiT

Transformer 架构如何实现反向过程的条件注入

Time Embedding 与条件注入

# DiT 条件注入（伪代码）
class AdaLN(layer):
    def forward(x, t_emb, c):
        # t_emb: 时间步的sinusoidal encoding
        # c: 额外的条件embedding
        gamma, beta = MLP(concat([t_emb, c]))
        # 残差自适应归一化
        return gamma * LN(x) + beta

def DiT_Block(x, t_emb, c):
    x = x + AdaLN(MHA(LN(x)), t_emb, c)
    x = x + AdaLN(FFN(LN(x)), t_emb, c)
    return x
AdaLN-Zero 的优势
DiT 进一步提出 AdaLN-Zero：初始化时 $\gamma=0$、$\beta=0$，使得每个残差块在训练初期等价于恒等映射，大幅加速收敛。这与 ResNet 的 shortcut 设计理念一脉相承。
